# Maze_Game 
Maze generator API based on randomized DFS algorithm

## Quick start
```Golang
func main() {
    // create new maze with height=10 and width=20
    maze := NewMaze(10, 20)
    maze.Print()
}
```
``` Console
$ go run main.go
```
